# Renovex - Construction & Architecture Website Clone

A frontend-only clone of the Renovex construction and architecture website demo. This is a fully responsive, modern website built with HTML, CSS, and JavaScript.

## Features

- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Modern and clean UI/UX
- ✅ Smooth scrolling navigation
- ✅ Animated counters and statistics
- ✅ Interactive FAQ accordion
- ✅ Project filtering system
- ✅ Contact form (frontend only)
- ✅ Newsletter subscription form
- ✅ Mobile-friendly hamburger menu
- ✅ Scroll-to-top button
- ✅ Preloader animation
- ✅ Fade-in animations on scroll

## File Structure

```
demo/
├── index.html          # Main HTML file
├── css/
│   └── style.css      # All styles and responsive design
├── js/
│   └── script.js      # JavaScript for interactivity
└── README.md          # This file
```

## Getting Started

1. Simply open `index.html` in your web browser
2. No build process or dependencies required
3. All styles and scripts are linked directly

## Customization

### Colors
Edit the CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #ff6b35;
    --secondary-color: #1a1a1a;
    /* ... other colors */
}
```

### Content
All content is in `index.html` - simply edit the text, links, and sections as needed.

### Images
Currently using gradient placeholders. Replace with actual images by:
1. Adding images to an `images/` folder
2. Updating the `background` or `src` attributes in the HTML/CSS
